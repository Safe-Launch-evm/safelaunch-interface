## Getting Started
To get started, follow the steps below:

1. Clone this repo
2. cd into any of the projects
3. Install the project's dependencies
4. Start development

``` bash
git clone https://github.com/Safe-Launch-evm/safe.launch.app.git
yarn install 
yarn dev
```
